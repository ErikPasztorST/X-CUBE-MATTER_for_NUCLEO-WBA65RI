STM_Provisioning example is used to encrypt and store in flash and securely the 
Device Attestation private key using PSA ITS.
It is part of the Factory Data provisioning procedure depicted in 
more details in the wiki:
https://wiki.st.com/stm32mcu/wiki/Connectivity:Matter_device_factory_data_provisioning

@par Keywords

Matter,DAC

@par Directory contents 

 
  Using Matter v1.4.0.0 SDK.
  
 
@par Hardware and Software environment

  - This example has been tested with an STMicroelectronics STM32WBA65I-DK1 board 
    
@par How to use it ? 
refer to wiki


Minimum requirements for the demo:
- 1 STM32WBA65I-DK1 board.


**** START DEMO ****
Follow the instruction defined in wiki


**** LED SUMMARY ****
- When DAC private Key is store with success in flash, the user can see the LED GREEN ON
- When DAC private Key storage failed the user can see LED GREEN blinking.
